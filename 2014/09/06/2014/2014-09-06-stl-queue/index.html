STL中优先队列的使用

STL中有一个优先队列的容器可以使用。

  • 头文件

queue 队列容器

vector 向量容器

  • 操作

优先级队列支持的操作

调用 功能
q.empty() 如果队列为空，则返回true，否则返回false
q.size() 返回队列中元素的个数
q.pop() 删除队首元素，但不返回其值
q.top() 返回具有最高优先级的元素值，但不删除该元素
q.push(item) 在基于优先级的适当位置插入新元素

对于Pascal留下来的手打堆习惯来说，其实对我用处不大，不过好像STL里面的复杂度更低，代码长度也能少点，以后尽量用STL好了。
